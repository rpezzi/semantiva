# EIR Examples

## In-process demo (EIR execution)

Run the minimal reference-suite demo that compiles `float_ref_01.yaml` to EIR and executes it in-process:

```bash
python -m examples.eir.run_float_ref_01_in_process
```

Expected output (scalar result): `3.0`.

This example demonstrates the core EIR scalar execution harness for developers working on EIRv1 compilation and execution.
